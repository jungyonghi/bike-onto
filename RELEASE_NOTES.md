# Bike Onto minimal repository

This repository is a minimized GitHub-ready portfolio packaging of the OBYBK / Bike Onto RAG inspection framework prototype. It is not presented as a production-ready service.

Included:
- README and referenced visual assets
- Core CLI/RAG source code
- BMO anchor blueprint and upper ontology seed generator
- Benchmark documentation and sample visual-inspector data
- Tiny reproducible data fixtures required by the tests
- Tests and dependency files

Excluded:
- raw/processed large datasets beyond the small committed fixtures
- local virtual environments
- external cloned references
- credentials, secrets, and local debug artifacts
